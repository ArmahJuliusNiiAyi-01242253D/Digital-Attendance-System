DIGITAL-ATTENDANCE-SYSTEM

Digital Attendance System (Week 1-4)implemented in C++.
This project is a Digital Attendance System developed in C++ as a Mid-Semester Capstone Project.
It replaces the manual method of taking attendance with a simple console-based system.
The program allows users to add and view registered students.
It supports creating attendance sessions for lectures.
Students can be marked present during each session.
The system can generate simple attendance reports and summaries.
It uses file handling to save and load data between program executions.
This project demonstrates practical use of Object-Oriented Programming and file handling in C++.
